# بازی دوز (Tic-Tac-Toe) با Tkinter

## اجرا

برای شروع بازی، کافیست فایل `run.bat` را در پوشه پروژه اجرا کنید.

> مطمئن شوید که Python روی سیستم شما نصب است.

## ساخت فایل اجرایی (EXE)

با اجرای `build.bat` می‌توانید نسخهٔ EXE بازی را بسازید.

## امکانات

دو حالت بازی: دو نفره و یک نفره با هوش مصنوعی ساده

تشخیص برد و مساوی

امکان شروع مجدد بازی

رابط گرافیکی ساده و کاربردی
